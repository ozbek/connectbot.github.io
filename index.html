ConnectBot

The first SSH client for Android.

GitHub

ConnectBot is a secure shell client for the Android platform. Its ultimate goal is to create a secure connection through which you can use a shell on a remote machine and transfer files back and forth to your phone.

Development Team

The current development team consists of Kenny Root, Paul Evans, and Torne Wuff.

All of us are on the #connectbot channel on Freenode if you have questions. You can join via Freenode’s webchat system or your own IRC client.

If you’re a developer, check out details about how the current codebase is designed.

Translations to your language

If you’d like to see ConnectBot translated into your language and you’re willing to help, then head on over to ConnectBot translations at Launchpad.
